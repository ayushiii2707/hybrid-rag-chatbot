# GitHub Social Preview Recommendations

This document outlines the recommended design specifications for the repository's GitHub Social Preview image.

## Dimensions
* **Resolution**: 1280 × 640 pixels (exactly 2:1 aspect ratio, optimized for GitHub card layouts)

## Suggested Layout
* **Left Panel**: Title block containing the name `Hybrid RAG Chatbot` in a clean, modern geometric sans-serif typeface (e.g., Outfit or Inter). Below it, place the main description and high-level bullet points.
* **Right Panel**: A stylized isometric wireframe diagram or abstract representation of the 3D WebGL particle node layout from the frontend UI.
* **Bottom Panel**: A neat row of logos or emblems for the core tech stack (FastAPI, React, Three.js, PostgreSQL, FAISS).

## Color Scheme
* **Background**: Sleek dark space background (#0B0F17) with a subtle radial gradient glowing from the center-right.
* **Accent Colors**: Electric blue (#00D2FF) and deep teal (#00F5D4) to mirror the 3D particle theme of the interface.
* **Text**: Crisp white (#FFFFFF) for the title and cool gray (#94A3B8) for descriptions.

## Typography
* **Main Title**: Bold Sans-Serif (e.g., Montserrat or Outfit), size 64pt.
* **Subtitle/Tags**: Regular Monospace (e.g., JetBrains Mono or Fira Code) for the tech stack tags, size 24pt.
