# Hybrid RAG Chatbot v1.0.0

We are pleased to release **Version 1.0.0** of the Hybrid RAG Chatbot repository, consolidating our production-ready backend, interactive 3D WebGL user interface, and evaluation benchmark suites.

## Highlights
- **Hybrid Retrieval (FAISS + BM25)**: Integrates high-performance dense semantic vector search with keyword-based sparse lexical matching.
- **Cross-Encoder Reranking**: Ensures high-fidelity matching and optimal positioning of retrieved contexts before passing them to the generator model.
- **FastAPI Backend**: Built on FastAPI featuring modular router endpoints, asynchronous event processing, and robust data persistence.
- **React + Three.js Frontend**: Rich, immersive 3D user experience using React Three Fiber, React Three Drei, and custom shader particles.
- **JWT Authentication**: Secure user session tracking and token-based state authorization.
- **OTP Verification**: Built-in OTP validation routing to safeguard registration and user onboarding pathways.
- **PostgreSQL Integration**: Direct SQLAlchemy support with pool configurations for database transactions.
- **Query Guardrails**: Added `QueryGuard` to intercept prompt injection patterns, anomalous inputs, and query hallucination hazards.
- **Benchmarking & Evaluation**: Integrated offline evaluators and diagnostic suites to measure retrieval scores and document chunking precision.
